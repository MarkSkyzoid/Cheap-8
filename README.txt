This is the readme for Cheap-8! Chip-8 Emulator created by Marco Vallario

INSTRUCTIONS:
Simply drag a rom on the Cheap-8 executable to run it!

Keys are mapped like this:

OUR KEYBOARD            CHIP-8 KEYBOARD
 
 1 2 3 4        =>       1 2 3 C
 Q W E R        =>       4 5 6 D
 A S D F        =>       7 8 9 E
 Z X C V        =>       A 0 B F

This program is open-source! 
You can use my source to make your own emulator and I'll be glad if you give me credits!

You can't sell this software!
It's FREE software and it must be downloaded freely!
